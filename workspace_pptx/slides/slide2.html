01 // THE PROBLEM

CURRENT STATE

Building Synaptic Command on stock LangGraph

An insurance multi-agent system: orchestrator + 8 subagents + persistent memory + project artifacts.

WHAT WE NEED

A central orchestrator that routes user intent to specialized subagents, recalls cross-conversation memory, and persists project outputs to Unity Catalog.

The naive approach: build it directly in LangGraph using StateGraph with a supervisor pattern and custom message reducers.

REQUIRED CAPABILITIES

  • Route to 8 domain subagents (dev, research, memory, 4× Genie)
  • Long-term memory across conversation threads
  • Persist project files to UC Volume
  • Stream tool calls + intermediate steps via SSE
  • Plan complex multi-step tasks reliably
